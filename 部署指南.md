### 第 4 步：启用 GitHub Pages ⏳

#### 操作步骤

1. **打开仓库页面**
   ```

   ```

2. **进入设置**
   - 点击 **Settings** 标签
   - 左侧菜单找到 **Pages**

3. **配置 Pages**
   ```
   Source: Deploy from a branch
   Branch: main
   Folder: / (root)
   ```

4. **保存**
   - 点击 **Save** 按钮

5. **等待部署**
   - 等待 1-2 分钟
   - 页面会自动刷新显示访问地址

```bash

echo "🔄 开始同步到 GitHub..."
cd ~/.openclaw/workspace-shangshu/portal

chmod +x ~/.openclaw/sync-portal.sh
```

### 设置定时任务

```bash
# 每小时同步一次
crontab -e

# 添加以下行

```

2. Custom domain: 输入你的域名

## 💡 常见问题

### Q1: 推送失败怎么办？

**错误：** `Permission denied (publickey)`

**解决：**
```bash
# 使用 HTTPS 而不是 SSH
git remote set-url origin https://github.com/你的用户名/openclaw-portal.git
git push
```

### Q2: Pages 不显示怎么办？

**解决：**
1. 检查仓库是否为 Public
2. 检查是否选择了 main 分支
3. 等待 5-10 分钟
4. 刷新页面

### Q3: 如何更新网站？

**方法：**
```bash
cd ~/.openclaw/workspace-shangshu/portal
./deploy.sh
```

### Q4: 如何删除网站？

**方法：**
1. 打开 GitHub 仓库
2. Settings → 滚动到底部
3. 点击 "Delete this repository"
