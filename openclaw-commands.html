🦞 OpenClaw 命令速查手册

EXFOLIATE! EXFOLIATE! — 随时调用的命令宝典

🚀 基础命令

openclaw --help

显示所有可用命令的帮助信息

openclaw --version

查看当前安装的 OpenClaw 版本

openclaw doctor

诊断系统配置问题，检查安全风险

推荐

⚙️ 安装与配置

openclaw onboard

启动交互式安装向导，引导完成初始配置

新手必备
openclaw onboard --install-daemon

安装守护进程（launchd/systemd 服务）

推荐
openclaw configure

手动配置 OpenClaw 参数

openclaw configure --section web

配置 Web 相关设置（如 Brave Search API）

🖥️ Gateway 管理

openclaw gateway

启动 Gateway 服务（前台运行）

openclaw gateway --port 18789

在指定端口启动 Gateway

openclaw gateway --verbose

以详细日志模式启动 Gateway

openclaw gateway status

查看 Gateway 运行状态

常用
openclaw gateway start

启动 Gateway 守护进程

openclaw gateway stop

停止 Gateway 守护进程

openclaw gateway restart

重启 Gateway 守护进程

🤖 Agent 交互

openclaw agent --message "你好"

向 Agent 发送消息并获取回复

常用
openclaw agent --message "任务" --thinking high

使用高思考级别执行复杂任务

openclaw agent --workspace /path/to/ws

在指定工作空间中运行 Agent

💬 消息管理

openclaw message send --to +1234567890 --message "Hello"

向指定号码/用户发送消息

openclaw message send --to @username --message "内容"

向指定用户发送消息（支持多平台）

📋 会话管理

openclaw sessions list

列出所有活跃的会话

常用
openclaw sessions list --active-minutes 30

列出最近 30 分钟活跃的会话

openclaw sessions history <sessionKey>

查看指定会话的历史记录

🔄 更新与维护

openclaw update

更新 OpenClaw 到最新版本

定期执行
openclaw update --channel stable

切换到稳定版频道并更新

可选频道：stable | beta | dev

npm install -g openclaw@latest

使用 npm 更新到最新版本

pnpm add -g openclaw@latest

使用 pnpm 更新到最新版本

📱 平台特定命令

openclaw macos install

安装 macOS 菜单栏应用

macOS
openclaw canvas open

打开 Live Canvas 可视化工作区

macOS
openclaw nodes list

列出所有已配对的节点设备

🔐 安全与配对

openclaw pairing list

列出所有待处理的配对请求

DM 必备
openclaw pairing approve <channel> <code>

批准配对请求（未知用户 DM 时使用）

安全必需
openclaw pairing reject <channel>

拒绝配对请求
